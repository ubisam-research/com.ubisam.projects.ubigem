## 🔑 1. 키락 정보

### 1.1 USB 타입 키락 1.0 (구형)
- 현재는 배포하지 않는 1.0 버전을 사용하는 업체용 키락
- USB 타입으로 탐색기에서 USB 드라이버로 인식됨

### 1.2 USB 형태 동글 키락 2.0 (신형)
- 신규 업체는 모두 2.0으로 배포
- 외형은 USB지만 실제 USB 장치는 아님

### 1.3 키락 미연결 시 3시간 주기 체크
- 3시간마다 키락 연결 여부 체크 후 강제 연결 해제
- 해제 시 `EXCP(예외)` 에러 발생
- 고객 로그에 EXCP가 있어도 키락 문제일 가능성 큼

### 1.4 키락 2.0 vs 1.0 구분 방법
#### 1) 설치 경로
- `"UbiSam.Net.KeyLock.Checker.exe"` 존재 → ✅ 키락 2.0  
  → 없으면 → ✅ 키락 1.0  
  → 단, 1.0은 신규 업체 배포 중단

#### 2) 키락 체크 방법
- 키락 2.0을 USB에 연결
- `"UbiSam.Net.KeyLock.Checker.exe"` 실행 후 정상 여부 확인

- `"UbiSam.GEM.Configuration.exe"`로 UGC 제작 시

# 🛠 Trouble Shooting

---

## 1. Disconnect 문제
- 최근 설비 제어 PC에 보안 프로그램 설치로 발생
  - 키락이 USB 장치임을 안내
  - 보안 프로그램 USB 잠금 해제 요청 필요
- USB 1.0의 경우 물리적 고장 가능성 있음
- 드물게 다른 USB 장치와 충돌 발생
