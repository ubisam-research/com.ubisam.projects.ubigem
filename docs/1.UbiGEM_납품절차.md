# UbiGEM / UbiGEM.PAC 납품 절차

> ※ UbiGEM과 UbiGEM.PAC은 **동일 절차**를 사용합니다.

---

## 1. 소프트웨어 다운로드 경로

### 1-1. UbiGEM 다운로드

- 사이트: [http://www.ubisam.com](http://www.ubisam.com)
- 경로:
  1. **커뮤니티** 메뉴 이동
  2. **자료실** 진입
  3. **1번 글** 선택  
     - 제목: **“ubiGEM관련 배포판”**

---

### 1-2. UbiGEM.PAC 다운로드

- 사이트: [http://www.ubisam.com](http://www.ubisam.com)
- 경로:
  1. **고객지원** 메뉴 이동
  2. **xxx 글** 선택  
     - (※ 현재 **홈페이지 리뉴얼 중**으로, 실제 게시글 명/위치는 변동될 수 있음)

---

## 2. KeyLock 납품 절차

1. 고객사에서 **UbiGEM 또는 UbiGEM.PAC 구매 요청** 접수
2. 담당자는 **본사 장석주 부장님 연락처**를 고객사에 전달
3. 고객사에서 **발주서 송부**
4. 발주서 수신 후,  
   **장석주 부장님이 KeyLock 생성 → 택배 발송**
